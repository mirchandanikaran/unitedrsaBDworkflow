How To Use Tender Intelligence

Quick guide for ingestion, filters, and interpretation of fields.

Back to Dashboard

1) Fetch Latest Data

Use Run Ingestion on the dashboard to pull new tenders from TED, CPPP, and World Bank.

2) Apply Filters

Use Country, Keyword, Source, and Status filters to narrow records.

Click Apply Filters to refresh results, or Clear to reset.

3) Read Metrics

  • Total Listed: records loaded for current filters.
  • Open: records currently marked open.
  • Closing Soon: records with closing date in next 7 days.
  • Source Breakdown: count by source in current filtered view.

4) Field Definitions

  • Organization: buyer/agency issuing the tender.
  • Country: country name/code as provided by source.
  • Status: normalized as open, closed, or awarded.
  • Closing Date: bid deadline when source provides it.
  • Source: ted, cppp, or worldbank.
  • Link: opens original notice page.
